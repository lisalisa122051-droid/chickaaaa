# WhatsApp Bot dengan Button List Support

Bot WhatsApp menggunakan library Baileys dengan fitur button list yang lengkap.

## Fitur
- ✅ Button List Message
- ✅ Support Grup (@lid format)
- ✅ Multi Prefix (!, /, ., #)
- ✅ Database JSON
- ✅ 30+ Commands
- ✅ Sticker Creator
- ✅ Downloader
- ✅ Fitur Grup
- ✅ Fun Games
